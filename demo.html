AI Chat Widget Demo

This is a demonstration of our embeddable AI chat widget. You should see the chat bubble in the bottom right corner.

Installation

To add this chat widget to your website, add the following script tag to your HTML:

<script src="https://kornfleyks.github.io/AI-Chatbot/widget.js" auto-init="true"></script>

Manual Initialization

If you prefer to initialize the widget manually, you can do so like this:

<script src="https://kornfleyks.github.io/AI-Chatbot/widget.js"></script>
<script>
  window.AIChatWidget.init();
</script>
